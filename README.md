<br/></br>### 课程介绍:<br/></br>C++ 是一种静态类型的、编译式的、通用的、大小写敏感的、不规则的编程语言，支持过程化编程、面向对象编程和泛型编程。
C++ 被认为是一种中级语言，它综合了高级语言和低级语言的特点。
C++ 是 C 的一个超集，事实上，任何合法的 C 程序都是合法的 C++ 程序。
